• To override any of the styles , please write your css style in a separate stylesheet and include it in the page after styles.css using the link tag.For e.g. if i want to change the default font size of body , i will write the following code in a separate css file say override.css and include in in the page . 
     
	
<link rel="stylesheet" type="text/css" href="Res/css/override.css"/>
<!--	----------- CODE inside override.css -------  -->
body{
	font-size: 24px;
}

• To start using this framework, we need to add the given stylesheet(styles.css) and javascript(global.js) along with few dependencies in the following manner. 
    				
<!DOCTYPE html>
	<html>
		<head>
		<title>Dubai Trade</title>
		<link rel="stylesheet" type="text/css" href="Res/css/styles.css"/>
		<script src="Res/js/vendor/modernizr-2.6.2.min.js" type="text/javascript"></script>
		<script src="Res/js/vendor/jquery-1.11.0.min.js" type="text/javascript"></script>
		<script src="Res/js/vendor/bootstrap.min.js" type="text/javascript"></script>
		</head>
		<body class="accolades">
			<div class="wrapper">
			lt;header class="header">
	<!---------------  Header will come here ------------------>
			</header>
			<section class="content">
	<!---------------  Main content will come here ------------------>
			</section>

	<!---------------  Footer will come here ------------------>
			</footer>
			</div>
			<script src="Res/js/global.js" type="text/javascript"></script>
		</body>
	</html>

    We used Bootstrap 3.3.5 and JQuery 1.12 for this framework development.

  • Browser Compatibility

    It works on all the modern browsers- Chrome, Firefox, Safari and IE 9+

  • Resolution

    • 1300px and higher for extra-large devices (eg- iMac)
    • 992 to 1299px for large devices (laptops)
    • 768 to 992px for medium devices (tablet and phablets)
    • 767px and below for small devices (mobile)

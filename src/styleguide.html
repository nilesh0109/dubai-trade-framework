Popup

For any modal popup , get the content from the div and use the following script functions .

DT.modalPopup.createPopup : to create the popup It take following four parameters: 1. jquery wrapped object of the div from where we need to get the data(URL of the iframe SRC in case of iframe content). 2. class name to be attached to the popup outermost div 3. (boolean parameter)do we need to open the popup content in an iframe . 4. (optional paramter) path to the ajax loader that needs to be shown while iframe content loads up .

DT.modalPopup.showPopup: to show the popup It takes the reference of the created popup as parameter .

DT.modalPopup.hidePopup : to hide the popup It takes the reference of the created popup as parameter .
